## Sistema di Tracking Clienti

✅ **Setup completato**: Dashboard clienti con aggiornamento automatico.

### Cosa è stato creato

**1. File master**: `Clients/Clienti.md`
- Tabella centralizzata di tutti i 10 clienti attivi
- Colonne: Cliente, Data Inizio, Consulenze Completate, Status
- Aggiornamento timestamp automatico

**2. Hook di sincronizzazione** (`.claude/settings.json`)
- Trigger: Ogni volta che salvi un file `Clients/{nome}/consulenze.md`
- Azione: Script bash che aggiorna il master

**3. Template per ogni cliente**: `Clients/{nome}/consulenze.md`
- Formato checkbox per tracciare sessioni completate: `- [x]` e `- [ ]`
- Tabella opzionale per metadati (data, argomento, durata)

### Come Usarlo

1. **Registra una nuova consulenza** in `Clients/{nome}/consulenze.md`:
   ```markdown
   - [x] 2026-07-15 — Sessione 5: Script Development
   - [ ] Sessione 6
   ```

2. **Salva il file** — il hook si attiva automaticamente

3. **Visualizza la dashboard**: `Clients/Clienti.md` è sempre aggiornato con il conteggio totale

### Struttura

```
Clients/
├── Clienti.md                 ← Master dashboard (aggiornato automaticamente)
├── egidio/
│   └── consulenze.md          ← Tracker sessioni Egidio
├── monica/
│   └── consulenze.md          ← Tracker sessioni Monica
├── beatrice/
│   └── consulenze.md
├── ... (altri 7 clienti)
```
