## 📝 ESERCIZIO 1: Le 5 Fasi del Prompt Perfetto

**Obiettivo:** Imparare il framework delle 5 fasi e testarlo con un prompt della tua nicchia.

### Fase 1: Definisci il Contesto (Chi Sei?)

Rispondi a questa domanda in base alla TUA specializzazione:

**Contesto Personale:**
```
Sono un farmacista che...
[SCRIVI QUI CHI SEI, CHE COSA FAI, CON CHI PARLI, QUALE SPECIALIZZAZIONE HAI]

Esempio: "Sono un farmacista specializzato in nutrizione funzionale per donne tra i 30-40 anni che soffrono di infiammazione cronica e disbiosi intestinale."

Il tuo contesto:
_______________________________________________________________
_______________________________________________________________
_______________________________________________________________
```

### Fase 2: Chiarisci l'Obiettivo (Cosa Vuoi Ottenere?)

Scrivi esattamente cosa vuoi che l'IA crei per te:

**Obiettivo Specifico:**
```
Voglio che tu mi crei...
[SCRIVI QUI ESATTAMENTE IL DELIVERABLE CHE TI SERVE]

Esempio: "Voglio che tu crei un articolo di 300 parole su come il magnesio e il probiotico specificoXXX aiutano a ridurre l'infiammazione intestinale."

Il tuo obiettivo:
_______________________________________________________________
_______________________________________________________________
_______________________________________________________________
```

### Fase 3: Identifica il Destinatario (Per Chi?)

Chi riceverà questo contenuto/risposta?

**Destinatario:**
```
Questo è per...
[SCRIVI QUI CHI LEGGERÀ/RICEVERÀ QUELLO CHE CREI]

Esempio: "Questo è per una donna di 35 anni senza background scientifico che soffre di gonfiore e ha provato antibiotici ma non ha funzionato."

Il tuo destinatario:
_______________________________________________________________
_______________________________________________________________
_______________________________________________________________
```

### Fase 4: Definisci Vincoli e Dettagli

Cosa deve avere (e NON avere) il tuo output?

**Vincoli Specifici:**
```
Il testo deve essere:
- Lunghezza: [max 200 parole / 500 parole / 1000 parole?]
- Tono: [professionale / caldo / semplice / diretto?]
- Formato: [articolo / tabella / PDF / lista?]
- NON deve contenere: [linguaggio troppo tecnico / gergo medico / liste puntate?]
- Deve includere: [fonti scientifiche / case study / esempi pratici?]

I tuoi vincoli:
_______________________________________________________________
_______________________________________________________________
_______________________________________________________________
```

### Fase 5: Descrivi il Formato Output

Come vuoi ricevere il risultato?

**Formato Output Finale:**
```
Voglio il risultato in formato:
- [ ] Documento Word
- [ ] PDF
- [ ] Testo semplice da copiare
- [ ] Tabella
- [ ] Altro: _________________

Il mio formato preferito:
_______________________________________________________________
```

## 🤖 ESERCIZIO 3: Testa il "Mega Prompter"

**Obiettivo:** Usare il Mega Prompter per farsi revisionare i tuoi prompt prima di usarli.

### Come Funziona:

1. Scrivi un prompt che vuoi testare
2. Lo dai al "Mega Prompter" (ChatGPT configurato con le regole d'oro)
3. Il Mega Prompter ti dice se il tuo prompt ha tutti gli elementi giusti
4. Tu lo migliori e lo usi

### Passaggi Pratici:

**Step 1 — Crea il Mega Prompter su ChatGPT:**

Copia e incolla questo nel Mega Prompter chat:

```
Tu sei il "Mega Prompter", un esperto nella creazione di prompt efficaci.

Il tuo compito è revisionare i prompt che ti invio e controllarli in base a questi criteri:

✅ CRITERI DI VALUTAZIONE:
1. CONTESTO CHIARO — Il prompt definisce chi è l'utente? (Sì/No)
2. OBIETTIVO SPECIFICO — Dice chiaramente COSA vuole ottenere? (Sì/No)
3. DESTINATARIO DEFINITO — Sa per chi è il contenuto? (Sì/No)
4. VINCOLI PRECISATI — Ha lunghezza, tono, formato definiti? (Sì/No)
5. OUTPUT DESCRITTO — Sa in quale formato ricevere il risultato? (Sì/No)

Quando mi invii un prompt, analizzalo e rispondimi così:
- Cosa funziona bene nel prompt
- Cosa manca o è vago
- 2-3 suggerimenti concreti per migliorarlo
- Versione migliorata del prompt (riscrivilo tu)

Inizia dicendomi: "Ho capito. Sono il Mega Prompter. Inviami i tuoi prompt e li analizzerò."
```
